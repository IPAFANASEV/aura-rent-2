# Aura Rent - Бизнес План

Интерактивный бизнес-план для сервиса аренды премиум-техники и оборудования для туристов.

## О проекте

Aura Rent — аренда техники и оборудования для туристов: GoPro, Ray-Ban Meta, JBL, SUP-борды.

- **Локация**: Крым (Ялта/Севастополь) → Санкт-Петербург
- **Стартовый капитал**: 500 000 ₽
- **Окупаемость**: ~4 месяца
- **Прогноз прибыли**: 1.5М ₽/год

## Технологии

- React 18
- TypeScript
- Tailwind CSS
- Recharts (графики)
- Vite

## Запуск локально

```bash
npm install
npm run dev
```

## Развертывание

Проект оптимизирован для развертывания на Vercel.
